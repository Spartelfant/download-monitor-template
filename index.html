Or 'about this document'

This file aims to provide some reference material to people creating custom templates (content-download-template.php) for the Download Monitor plugin for WordPress. I created this page after I was unable to find any documentation by its authors that was to my liking. I based this document on the DLM_Download and DLM_Download_Version classes from version 4.4.2 (released on 20190403) of the plugin.
